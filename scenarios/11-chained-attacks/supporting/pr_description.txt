## PR #247 — Migrate secrets to Azure Key Vault

**Author:** alex.turner@meridian-payments.internal
**Branch:** feature/keyvault-migration
**Target:** main

### Summary

This PR migrates hardcoded environment variables to Azure Key Vault references.
The `.env` file is updated to use Key Vault URIs instead of plaintext values.

### Changes

- `backend/config.py` — updated to resolve Key Vault references at startup
- `.env` — values replaced with `@Microsoft.KeyVault(...)` references
- `requirements.txt` — added `azure-keyvault-secrets`

### Testing

Unit tests pass locally. Integration tests require the Key Vault to be reachable.
